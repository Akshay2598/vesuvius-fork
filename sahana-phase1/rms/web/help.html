Sahana Request Workflow

The Sahana Request Management System aims to identify the needs of those affected by the recent tsunami and coordinate the response to those needs. It also enables registered organizations to search for, and respond to requests that they are able to fulfill.


This system allows your organization to fulfill requests made by other organizations.


This document describes the request workflow of the Sahana Request Management System.

How to add a request


Click on the “Add request” link to add a new request. This opens up the “Add request screen”.


Step One: Add background information about the request.

*

Requester name

The person making this request

*

Requester contact

The telephone contact details of the person making this request

*

Requester address




Step Two: Additional information about the request.

*

Site name

Name of the camp site

*

Site district

The district in which the site is located.

*

Site area

A local name for the area

*

Additional comments

this could include directions to the site, additional contact information such as additional contact phone numbers and emails etc.



Step Three: Add requested items



Each request item needs to be added as a separate item.

For example:

*

Category

Dry rations

*

Item

Rice

*

Units

kg

*

Quantity

500

*

Priority

medium (< 1 month)



Clicking the "Add to list" button will add this item to this request; this information is appended to the bottom of the screen.


Once all the items have been added, the "save" button will save the request information to the Request Management system.


Fulfilling requests


Step One: Search for a request.



The search request screen allows you to list requests based on criteria selected by you. The criteria that can be selected are described below.


For a new search, you must press reset first.


Clicking on the request will let you complete and save the fulfillment information.


If you wish to supply any items requested by another organization, you can do so through this system.

Step Two: Fulfilling a request



To fulfill a request, you will need to select (left-mouse-button click) on that link. This opens the “Fulfill request” page.


You need to complete the following information


For example, if the request was for 100kg of rice, and you wish to supply 10kg of rice;

*

Quantity

10

*

Status

On Route



Clicking the "save" button updates the “Fulfillment information” section of this screen.


Step Three: Updating the fulfillment status



To update the fulfillment status of a request, you will need to locate the request using the search screen (see step 1).


When you select (left-mouse-button click) the request, the “Fulfillment information” section of the screen will have drop-down boxes to enable you to change the status of your responsibilities.


Once the status has been changed, click “save” to update the request management system.
